Module colors

Colors library module.

Also serves as the metatable for tailor-color objects. Metatable functions are written separately in the meta.tailor-color module doc.

Contains functions to interface with colors.

Color objects have the typename tailor-color.

Library functions

fromHex(hexString)
Returns a tailor-color object with a color corresponding to the color defined by the given hex string.

Parameters:

  • hexString string the hex string to derive the color from, in the format of #RRGGBB

Returns:

    a tailor-color object of the specified color
fromRGB(r, g, b)
Returns a tailor-color object with a color corresponding to the color defined by the given RGB values

Parameters:

  • r int the red value of the color, between 0 and 255
  • g int the green value of the color, between 0 and 255
  • b int the blue value of the color, between 0 and 255

Returns:

    a tailor-color object of the specified color values
