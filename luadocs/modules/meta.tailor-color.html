Functions

color:getRGB()
Returns the RGB values of this tailor-color.

Returns:

  1. the red value of this color
  2. the green value of this color
  3. the blue value of this color
color:setRGB(r, g, b)
Sets the RGB values of this tailor-color.

Parameters:

  • r int the red value of the color, between 0 and 255
  • g int the green value of the color, between 0 and 255
  • b int the blue value of the color, between 0 and 255
