Module base

Base library module.

The functions within this module are installed within the global table of any script loaded by Undertailor; that is, they can be executed simply without prefixing the function name with "base."

local blah = nil
local api = loadscript("scripts/lol.lua")

Internally, this module also remove functions from the global table that may potentially be harmful to the host machine, or functions whose functionality have been replaced by Undertailor, including:

  • load
  • dofile
  • loadfile
  • package
  • collectgarbage
  • all of the os library

Functions

loadscript(scriptPath) Loads the script at the given path, starting from the root directory where Undertailor was started from (where main.lua is), and returns a table of the functions contained.
execute(scriptPath) Loads the script at the given path, starting from the runtime directory of Undertailor (where main.lua is), and executes it.


Functions

loadscript(scriptPath)
Loads the script at the given path, starting from the root directory where Undertailor was started from (where main.lua is), and returns a table of the functions contained.

Parameters:

  • scriptPath string the path denoting the script file to load starting from the run directory

Returns:

    a table with functions loaded from the target script

Usage:

         local someApi = loadscript("scripts/myapi.lua")
     someApi.myFunction() -- defined as "function myFunction()" in myapi.lua
execute(scriptPath)
Loads the script at the given path, starting from the runtime directory of Undertailor (where main.lua is), and executes it.

Parameters:

  • scriptPath string the path denoting the script file to load starting from the run directory

Usage:

        -- runs the lua code in thegreatpapyrus.lua
    execute("scripts/scenes/thegreatpapyrus.lua")
