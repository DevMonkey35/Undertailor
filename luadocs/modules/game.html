Module game

Game library module.

Contains functions that can change the base game.

Contains submodules:

  • game.animation
  • game.audio
  • game.control (currently unused)
  • game.environment
  • game.graphics

Functions

setWindowTitle(title)
Sets the title of the game window.

Parameters:

  • title string the new title of the game window, or nil to use the default
setFrameCap(cap)
Sets the frame limit of the game.

The game will only allow itself to cap at a minimum of 30fps. Giving a lower value will simply have the value resolve to 30.

Parameters:

  • cap int the maximum framerate the game is allowed to achieve
