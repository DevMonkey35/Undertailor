Module game.audio

Audio library module.

Also serves as the metatable for tailor-audio objects. Metatable functions are written separately in the meta.tailor-audio module doc.

Contains functions to interface with audio files.

Sounds have the typename tailor-audio-sound. Music tracks have the typename tailor-audio-music.

Object retrieval functions

getMusic(id) Returns the tailor-audio-music object of the given ID.
getSound(id) Returns the tailor-audio-sound object of the given ID.

Global volume control functions

getMasterVolume() Gets the current volume of the entire game.
setMasterVolume(volume) Sets the current volume of the entire game.
getMusicVolume() Gets the current volume of music played by the game.
setMusicVolume(volume) Sets the current volume of music played by the game.
getSoundVolume() Gets the current volume of sounds played by the game.
setSoundVolume(volume) Sets the current volume of sounds played by the game.

Global stop functions

stopAllAudio() Kills all currently playing audio; both sound and music.
stopAllMusic() Kills all currently playing music.
stopAllSounds() Kills all currently playing sound.


Object retrieval functions

getMusic(id)
Returns the tailor-audio-music object of the given ID.

Parameters:

  • id string the id of the target music file

Usage:

         local asgore = game.audio.getMusic("fight.mus_vsasgore")
     -- points to music/fight/mus_vsasgore
     --                                    .ogg,
     --                                    .wav, or
     --                                    .mp3
getSound(id)
Returns the tailor-audio-sound object of the given ID.

Parameters:

  • id string the id of the target music file

Usage:

         local talkovw = game.audio.getSound("ds_ovw")
     -- points to sounds/ds_ovw
     --                         .ogg,
     --                         .wav, or
     --                         .mp3

Parameters:
