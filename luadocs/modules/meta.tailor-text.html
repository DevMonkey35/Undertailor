Functions

text:addComponent(component)
Adds a single text component to this parent tailor-text object.

Parameters:

  • component tailor-textcomponent the component to add
text:addComponents(...)
Adds all the given text components to this parent tailor-text object, in the order given.

Parameters:

  • ... tailor-textcomponent the components to add to this parent text object

Usage:

         local myText;
     local myComponent1;
     local myComponent2;
     local myComponent3;
     -- assume these variables are filled with their respective values

     myText:addComponents(myComponent1, myComponent2, myComponent3)
     -- an infinite amount of parameters may be supplied
text:getComponentAtCharacter(charIndex)
Returns the tailor-textcomponent owning the character at the given index. The first character is at index 1.

Parameters:

  • charIndex int the index of the target character

Returns:

    the tailor-textcomponent owning the target character

Usage:

         local text; -- this text has "ham" and "burgers" = "hamburgers"
     text:getComponentAtCharacter(1) -- returns the "ham" component
     text:getComponentAtCharacter(3) -- also returns ham
     text:getComponentAtCharacter(4) -- returns the "burgers" component
text:getMembers()
Returns an array containing all the tailor-textcomponent objects parented by this tailor-text object in the order they were registered.

Returns:

    an array of all this text's textcomponents
