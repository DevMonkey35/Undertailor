Module game.animation

Animation library module.

Also serves as the metatable for tailor-animation objects. Metatable functions are written separately in the meta.tailor-animation module doc.

Contains functions to interface with animations.

Animations have the typename tailor-animation.

createAnimation(setName, animName)
Creates new usable animation data targetting the given animation.

Note that this is not meant to be a repetitively-used function; this function creates new data every time it is called; creating animation data for the same animation will not return the same animation. It is intended to create animation local to a single object that wants to use it.

Parameters:

  • setName string the name of the set where the animation is registered
  • animName string the name of the animation within the set

Returns:

    a new tailor-animation object referencing the given animation

Usage:

         local anim = game.animation.createAnimation("frisk", "walk_up")
     local anim2 = game.animation.createAnimation("frisk", "walk_up")
     -- these two are not the same

     anim.setRuntime(1.2)
     anim:play()
     anim2:play()

     -- these two run separately
