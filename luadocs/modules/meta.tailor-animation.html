Functions

animation:getRuntime()
Returns the amount of time that's passed for this animation since starting.

The value returned is in seconds, and is not limited by the length of the animation. Runtime continues even if the actual animation has no more frames if it isn't looping, or continues forever while its looping.

Returns:

    how long the animation has been running in seconds

Usage:

         local anim -- our animation
     local runtime = anim:getRuntime()
animation:setRuntime(time)
Sets the amount of time that's passed for this animation, in seconds.

If the animation has not started, this function will set the runtime to the given time and pause. The effect is the same for if the animation is currently paused. If the animation is currently running, the runtime is set and playback of the animation continues from the set point.

Parameters:

  • time number the runtime to set for this animation, in seconds

Usage:

         local anim -- our animation
     anim:setRuntime(12.04) -- this animation has now been playing for 12.04 seconds
animation:isPlaying()
Returns whether or not this animation is playing, that is; not paused and has already started.

Returns:

    whether or not the animation is playing

Usage:

         local anim -- our animation
     local isPlaying = anim:isPlaying()
animation:isLooping()
Returns whether or not this animation is looping. By default, when animation data is generated, this value is set to whatever the reference animation has specified in its definition (if the animation said it looped, by default animation data generated for it is looped).

Returns:

    whether or not this animation loops playback

Usage:

         local anim -- our animation
     local isLooping = anim:isLooping()
animation:setLooping(flag)
Sets whether or not this animation should be looping.

Looping animations will continually repeat their playback forever, non-looping animations will stop at their last frame.

Parameters:

  • flag bool new state of looping
animation:play()
Starts this animation's playback from the very beginning.

This function will always start playback from the start of the animation. If one wishes to change initial playback point, the animation:setRuntime function can be used in tandem with animation:resume.

Usage:

         local anim -- our animation
     anim:play()
animation:pause()
Pauses this animation's playback, keeping runtime in memory to resume playback at a future point.

Typically used with animation:resume to stop/start animations.

animation:stop()
Stops this animation's playback.

This stops playback of the animation as well as resets it to the default state of not playing at the initial position of the animation, as opposed to animation:pause which simply freezes the animation at its current runtime.

animation:resume()
Resumes this animation at its current runtime.

If the animation has yet to start, this function invokes the animation:play function to start the animation, otherwise it resumes the animation starting at its currently recorded runtime.

animation:drawCurrentFrame(x, y[, scale=1.0][, rotation=0.0])
Draws the current frame of this animation with the specified parameters.

Parameters:

  • x number the x position to draw at
  • y number the y position to draw at
  • scale number the scale to draw the frame at (default 1.0)
  • rotation number the rotation to draw the frame at in degrees (default 0.0)
