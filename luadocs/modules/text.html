Module text

Text library module.

Also serves as the metatable for tailor-text objects. Metatable functions are written separately in the meta.tailor-text module doc.

Contains submodules:

  • text.component

Contains functions to interface with text objects used by Undertailor.

Text objects have the typename tailor-text.

newDisplayMeta([offX=0][, offY=0][, scaleX=1][, scaleY=1][, color=nil])
Generates new display metadata to be used by styles in order to customize text rendering.

Parameters:

  • offX number the horizontal positional offset of the target sprite (default 0)
  • offY number the vertical positional offset of the target sprite (default 0)
  • scaleX number the horizontal scale of the target sprite (default 1)
  • scaleY number the vertical scale of the target sprite (default 1)
  • color tailor-color the color of the target sprite (default nil)
newText(fontname[, stylename=nil][, color=nil][, sound=nil][, playspeed=35][, segmentsize=1][, delay=0])
Creates a new parent text object using the given parameters.

Parameters:

  • fontname string the name of the font to render the components of this text with by default
  • stylename string the name of the style to render the components of this text with by default (default nil)
  • color tailor-color the color to render the components of this text with by default (default nil)
  • sound tailor-audio-sound the sound to play while rendering the components of this text object, by default (default nil)
  • playspeed int the speed at which the components of this text with by default, in characters per second (default 35)
  • segmentsize int the segment size for characters of this text (default 1)
  • delay number the time in seconds to wait before a text renderer should continue on to another text object after finishing this text (default 0)
drawText(text, x, y[, scaleX=1.0][, scaleY=1.0][, alpha=1.0])
Draws the given text using the specified parameters.

Parameters:

  • text tailor-text the text to draw
  • x number the x position to draw at
  • y number the y position to draw at
  • scaleX number the horizontal scale of the text (default 1.0)
  • scaleY number the vertical scale of the text (default 1.0)
  • alpha number the alpha value of the text (default 1.0)
