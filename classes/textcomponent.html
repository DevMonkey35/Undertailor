Class textcomponent

textcomponents are the individual segments of text in a `text object.

Note that textcomponents are immutable.

Methods

textcomponent:getText() Returns the text contained by this textcomponent.
textcomponent:getFont() Returns the font used to render this textcomponent.
textcomponent:getColor() Returns the color used to render this textcomponent.
textcomponent:getSound() Returns the sound played for each segment of this textcomponent whenever it gets rendered.
textcomponent:getStyles() Returns an array of text styles used when rendering this textcomponent.
textcomponent:getTextSpeed() Returns the speed at which the text in this segment is played, in characters per second.
textcomponent:getSegmentSize() Returns how many characters are in each segment.
textcomponent:getDelay() Returns the amount of time to wait before rendering the text in this textcomponent, in seconds.
