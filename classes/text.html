Class text

text objects are databags holding a series of characters and parameters to form fully-customizable text objects.

This class also holds the functions of the following classes:

  • renderable

Methods

text:getBoundedText()
Returns the text bounded between this text's string bound indices.

See text:getStringBounds for a brief description on string boundaries.

Returns:

    this text's currently bounded text
text:getComponents()
Returns an array containing each component held by this text object.

Returns:

    this text's current component array, read-only
text:getSpaceTaken()
Returns the space taken by this text in pixels, when rendered on a normal scale.

String boundaries are taken into account.

Returns:

  1. the width of the text
  2. the height of the text
text:getStringBounds()
Returns the current string boundaries set upon this text.

String boundaries are a performance-friendly way of isolating certain segments of text within a text object. This serves as an alternative to the text:substring function, which generates a new text object on every call, which takes more time and memory than simply modifying the scope of the text, which this function does.

This function should always be used during tasks like rendering, where the object is frequently queried on what to write next.

If both bound values are -1, then it is assumed that no string boundaries are set, allowing text:getBoundedText to return the full contained text, similar to the function of text:getText.

Returns:

  1. the left bound, or -1 if not set (defaulting to 1)
  2. the right bound, or -1 if not set (defaulting to the length of the text)
text:setStringBounds(first, second)
Sets the string boundaries for this text.

See text:getStringBounds for a brief description on string boundaries.

Parameters:

  • first int the left bound, or -1 to default
  • second int the right bound, or -1 to default
text:getTextComponentAt(index)
Returns the textcomponent located at the provided index in this text.

Parameters:

  • index int the character index to query
text:substring(first, second)
Clones this text, including only the text found within the provided string boundaries.

Do not use this function repetitively in somewhere like a render function, as creating text over and over is not performance-friendly and makes our garbage collector unhappy. The text:setStringBounds function is much better for such tasks.

Parameters:

  • first int the left bound
  • second int the right bound
