Class scheduler

schedulers are one of the three main subsystems running under an environment.

Used to schedule tasks to occur every frame in the current environment, and is the first to be processed, before the uicontroller.

This class also holds the functions of the following classes:

  • subsystem
  • processable
  • destructible

Methods

scheduler:registerTask(task[, active=false]) Registers the provided task implementation into this scheduler.
scheduler:cancelTask(id) Cancels the task with the provided ID.
scheduler:hasTask(id) Returns whether or not the task of the provided ID is currently being ran by this scheduler.
