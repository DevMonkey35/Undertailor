Class pressdata

pressdata objects are databags for the individual state of input for specific keyboard keys, typically managed by an inputdata instance.

Methods

pressdata:getLastReleaseTime() Returns the amount of time that had passed since this key was last released, in seconds.
pressdata:justReleased([time=0]) Returns whether or not this key had been released within the last given seconds of time.
pressdata:getLastPressTime() Returns the amount of time that had passed since this key was last pressed, in seconds.
pressdata:justPressed([time=0]) Returns whether or not this key had been pressed within the last given seconds of time.
pressdata:getHoldTime() Returns how long this key has been held for if it is currently pressed.
pressdata:isPressed() Returns whether or not this key is currently pressed.


Methods

pressdata:getLastReleaseTime()
Returns the amount of time that had passed since this key was last released, in seconds.

Returns:

    how long its been since this key was released, in seconds
pressdata:justReleased([time=0])
Returns whether or not this key had been released within the last given seconds of time.

If 0 is passed for time, this function returns whether or not the key had been released in the current frame.

Parameters:

  • time number the time frame to search in (default 0)

Returns:

    whether this key had been released within the last given seconds, or within the current frame
pressdata:getLastPressTime()
Returns the amount of time that had passed since this key was last pressed, in seconds.

Returns:

    how long its been since this key was pressed, in seconds
pressdata:justPressed([time=0])
Returns whether or not this key had been pressed within the last given seconds of time.

If 0 is passed for time, this function returns whether or not the key had been pressed in the current frame.

Parameters:

  • time number the time frame to search in (default 0)

Returns:

    whether this key had been pressed within the last given seconds, or within the current frame
pressdata:getHoldTime()
Returns how long this key has been held for if it is currently pressed. If not currently pressed, returns how long it was held for the last time it was pressed, or 0 if it was never pressed prior to calling this function.

Returns:

    how long this key has, or had been pressed for, or 0 if never pressed
pressdata:isPressed()
Returns whether or not this key is currently pressed.

Returns:

    if this key is currently pressed
