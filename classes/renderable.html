Class renderable

renderable objects have a function specifically for drawing onto the screen.

Methods

renderable:getTransform() Returns the transform of this renderable.
renderable:setTransform(transform) Sets the transform of this renderable.
renderable:draw([x=0.0][, y=0.0][, transform=renderable:getTransform]) Draws this renderable onto the screen.


Methods

renderable:getTransform()
Returns the transform of this renderable.

Returns:

    the transform of this renderable
renderable:setTransform(transform)
Sets the transform of this renderable.

If a renderable possesses a transform, it should never allow its transform to become nil. Implementations should query the default transform to duplicate its values into this renderable's current transform using game.graphics.getDefaultTransform should the passed transform be nil.

Parameters:

  • transform transform the new transform to set
renderable:draw([x=0.0][, y=0.0][, transform=renderable:getTransform])
Draws this renderable onto the screen.

Parameters:

  • x number the x coordinate to draw at (default 0.0)
  • y number the y coordinate to draw at (default 0.0)
  • transform transform the transform to draw with (default renderable:getTransform)
