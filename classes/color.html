Class color

color objects are data representations of colors.

Tailor holds colors stored in RGB format.

Methods

color:getRGB() Returns the RGB values stored within this color.
color:set(r, g, b) Sets the RGB values stored within this color.
color:add(r, g, b) Adds to the RGB values stored within this color.
color:multiply(r, g, b) Multiplies the RGB values within this color.
color:subtract(r, g, b) Subtracts from RGB values within this color.
color:clone() Creates and returns a copy of this color.
color:toHex() Returns the value of this color in hex color format (#RRGGBB).


Methods

color:getRGB()
Returns the RGB values stored within this color.

Values are between 1 and 255, inclusive.

Returns:

  1. the R value of the color
  2. the G value of the color
  3. the B value of the color
color:set(r, g, b)
Sets the RGB values stored within this color.

Values are between 1 and 255, inclusive.

Parameters:

  • r int the new red value of the color
  • g int the new green value of the color
  • b int the new blue value of the color
color:add(r, g, b)
Adds to the RGB values stored within this color.

Values are between 1 and 255, inclusive.

Parameters:

  • r int the red value to add
  • g int the green value to add
  • b int the blue value to add
color:multiply(r, g, b)
Multiplies the RGB values within this color.

Parameters:

  • r int the red value to multiply by
  • g int the green value to multiply by
  • b int the blue value to multiply by
color:subtract(r, g, b)
Subtracts from RGB values within this color.

Parameters:

  • r int the red value to subtract by
  • g int the green value to subtract by
  • b int the blue value to subtract by
color:clone()
Creates and returns a copy of this color.

Returns:

    a clone of this color
color:toHex()
Returns the value of this color in hex color format (#RRGGBB).

Returns:

    the hex representation of this color
