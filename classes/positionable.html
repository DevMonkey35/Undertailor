Class positionable

positionable objects hold their position within their containers.

Methods

positionable:getPosition() Returns the position of this positionable.
positionable:setPosition(x, y) Sets the position of this positionable.
positionable:getHeight() Returns the height of this positionable.
positionable:setHeight(height) Sets the height of this positionable.


Methods

positionable:getPosition()
Returns the position of this positionable.

Returns:

  1. the x coordinate of this positionable
  2. the y coordinate of this positionable
positionable:setPosition(x, y)
Sets the position of this positionable.

Parameters:

  • x the new x coordinate of this positionable
  • y the new y coordinate of this positionable
positionable:getHeight()
Returns the height of this positionable.

Height determines the Y offset for a positionable object in the Overworld, as objects in the Overworld are usually rendered in an order based on their current Y position. To allow objects to have an illusion of altitude without needing to go to an unnecessarily high Y value in order to avoid an object jumping in front of it, height allows the visual Y position of an object to be offset without disturbing the render order.

Height is only used by world objects. This function returns 0 on any other type.

Returns:

    the height of this positionable
positionable:setHeight(height)
Sets the height of this positionable.

See positionable:getHeight for an explanation on height.

Parameters:

  • height number the new height of this positionable
