Class uiobject

uiobject are the main container objects for uicomponents, used in the UI layer.

This class also holds the functions of the following classes:

  • eventlistener
  • identifiable
  • positionable
  • destructible
  • processable
  • renderable

Methods

uiobject:isActive() Returns whether or not this uiobject is active.
uiobject:getMaxLifetime() Returns the maximum lifetime this uiobject will live upon registration, in seconds.
uiobject:setMaxLifetime(seconds) Sets the maximum lifetime this uiobject will live upon registration, in seconds.
uiobject:isPastLifetime() Returns whether or not this uiobject is past is lifetime.
uiobject:getLifetime() Returns the current lifetime of this uiobject, in seconds.
uiobject:resetLifetime() Resets the current lifetime of this uiobject to 0.
uiobject:registerComponent(component) Registers the provided uicomponent into this uiobject.

Methods

uiobject:isActive()
Returns whether or not this uiobject is active.

Returns:

    if this uiobject is active
uiobject:getMaxLifetime()
Returns the maximum lifetime this uiobject will live upon registration, in seconds. A value of 0 or less indicates an infinite lifetime.

Returns:

    the maximum lifetime of this uiobject in seconds
uiobject:setMaxLifetime(seconds)
Sets the maximum lifetime this uiobject will live upon registration, in seconds.

Parameters:

  • seconds number the new max lifetime for this uiobject, in seconds
uiobject:isPastLifetime()
Returns whether or not this uiobject is past is lifetime. This usually is only true for an object that had already been removed from a uicontroller.

Returns:

    if this uiobject is past its max lifetime, and is scheduled for removal
uiobject:getLifetime()
Returns the current lifetime of this uiobject, in seconds. This initially represents time since registration to a uicontroller, however the uio:resetLifetime function can be called to reset lifetime back to 0.

Returns:

    the current lifetime of this uiobject in seconds
uiobject:resetLifetime()
Resets the current lifetime of this uiobject to 0.
uiobject:registerComponent(component)
Registers the provided uicomponent into this uiobject.

Parameters:

  • component uicomponent the uicomponent to register
uiobject:remove()
Removes this uiobject from processing.

This function differs from calling destructible:destroy in that it allows the uiobject to be reusable.
