Class font

font objects store a set of sprites used to display language in-game.

Methods

font:getLineSize() Returns the distance between two lines of text rendered with this font, in pixels.
font:getLetterSpacing(character) Returns the letter spacing for the provided character, when rendered with this font.
font:getCharacterList() Returns a string containing all characters supported by this font.
font:getCharacterSprite(character) Returns the sprite for the provided character when using this font.


Methods

font:getLineSize()
Returns the distance between two lines of text rendered with this font, in pixels.

Returns:

    the size of lines with this font, in pixels
font:getLetterSpacing(character)
Returns the letter spacing for the provided character, when rendered with this font.

Characters past the very first one in the string provided will be ignored.

Parameters:

  • character string the character to query with

Returns:

  1. the amount of empty pixels guaranteed to be behind the provided character
  2. the amount of empty pixels guaranteed to be in front of the provided character
font:getCharacterList()
Returns a string containing all characters supported by this font.

Returns:

    a string of supported characters
font:getCharacterSprite(character)
Returns the sprite for the provided character when using this font.

Parameters:

  • character

Returns:

    the provided character's sprite, or nil if none
