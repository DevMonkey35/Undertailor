Class audiodata

audiodata objects hold audio clips with parameters that may be controlled, such as volume, pitch, and pan.

Methods

audiodata:getVolume() Returns the volume set for this audiodata.
audiodata:setVolume(volume) Sets a volume of this audiodata.
audiodata:getPitch() Returns the pitch value set for this audiodata.
audiodata:setPitch(pitch) Sets the pitch value of this audiodata.
audiodata:getPan() Returns the panning value set for this audiodata.
audiodata:setPan(pan) Sets the panning value of this audiodata.
audiodata:isLooping() Returns whether or not the audio of this audiodata is looping.
audiodata:setLooping(looping) Sets whether or not the audio of this audiodata is looping.
audiodata:isPlaying() Returns whether or not the audio of this audiodata is currently playing.
audiodata:stop() Stops the audio associated with this audiodata from playing.


Methods

audiodata:getVolume()
Returns the volume set for this audiodata.

Volumes are between 0.0 and 1.0.

Returns:

    the volume of this audiodata
audiodata:setVolume(volume)
Sets a volume of this audiodata.

The provided value is automatically bounded between 0.0 and 1.0.

Parameters:

  • volume number the new volume
audiodata:getPitch()
Returns the pitch value set for this audiodata.

Pitch values are between 0.5 and 2.0. Note that pitch also affects speed.

Returns:

    the pitch of this audiodata
audiodata:setPitch(pitch)
Sets the pitch value of this audiodata.

The provided value is automatically bounded between 0.5 and 2.0. Use 1.0 for default pitch/speed.

Parameters:

  • pitch number the new pitch
audiodata:getPan()
Returns the panning value set for this audiodata.

Panning values are between -1.0 and 1.0.

Returns:

    the pan of this audiodata
audiodata:setPan(pan)
Sets the panning value of this audiodata.

The provided value is automatically bounded between -1.0 and 1.0. Use 0.0 to center audio. Negative values allocate sound to the left speaker, positives to the right.

Parameters:

  • pan number the new pan
audiodata:isLooping()
Returns whether or not the audio of this audiodata is looping.

Returns:

    if this audiodata is looping
audiodata:setLooping(looping)
Sets whether or not the audio of this audiodata is looping.

Parameters:

  • looping whether this audiodata should loop
audiodata:isPlaying()
Returns whether or not the audio of this audiodata is currently playing.

Returns:

    if this audiodata is playing
audiodata:stop()
Stops the audio associated with this audiodata from playing.
