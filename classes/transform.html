Class transform

transform objects are databag classes used to dictate how a renderable object should be drawn.

Methods

transform:getScaleX() Returns the horizontal scale to draw with.
transform:getScaleY() Returns the vertical scale to draw with.
transform:getFlipX() Returns whether or not the rendered object should be flipped horizontally.
transform:getFlipY() Returns whether or not the rendered object should be flipped vertically.
transform:getRotation() Returns the rotation to draw with, in degrees.
transform:setScaleX(scale) Sets the horizontal scale to draw with.
transform:setScaleY(scale) Sets the vertical scale to draw with.
transform:setFlipX(flip) Sets whether or not the rendered object should be flipped horizontally.
transform:setFlipY(flip) Sets whether or not the rendered object should be flipped vertically.
transform:setRotation(deg) Sets the rotation to draw with, in degrees.
