Module game

game library for interfacing with core functionality.

Sublibraries

graphics game.graphics library used for controlling how objects are rendered and for shape rendering.
audio game.audio library used for retrieving and controlling instances of audio.
control game.control library used to query user input.

Environment functions

getEnvironment(name)
Returns the environment under the given name, or creates a new one if it doesn't exist.

Parameters:

  • name string the name of the environment

Returns:

    the environment under the given name, never nil
destroyEnvironment(env)
Destroys the given environment.

Alternatively, one can use environment:destroy.

Parameters:

  • env environment the environment to destroy
destroyEnvironment(name)
Destroys the environment of the given name, if it exists.

Parameters:

  • name string the name of the environment to destroy
getActiveEnvironment()
Returns the current active environment, if available.

Returns:

    the current active environment, or nil if not set
setActiveEnvironment(env)
Sets the active environment.

Parameters:

  • env environment the environment to set as active, or nil to disable
setActiveEnvironment(name)
Sets the active environment.

Parameters:

  • name string the name of the environment to set as active

Object creation functions

The three primary implementable types, worldobject, uicomponent, and renderable can be called without providing a script path for external implementation.
newWorldObject([scriptPath=nil][, ...=nil])
Creates a new worldobject, implemented by the script at the given path.

Parameters:

  • scriptPath string the path to the implementing script (default nil)
  • ... vargs creation parameters, passed to the implementing script's create function (default nil)

Returns:

    a worldobject
newWorldRoom(scriptPath[, ...=nil])
Creates a new worldroom, implemented by the script at the given path.

Parameters:

  • scriptPath string the path to the implementing script
  • ... vargs creation parameters, passed to the implementing script's create function (default nil)

Returns:

    a worldroom

    a uicomponent
