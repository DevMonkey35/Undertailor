Library functions

fromHex(hex)
Creates a color matching the provided hex value.

Hex values are in #RRGGBB format.

Parameters:

  • hex string the hex value to create a color with
fromRGB(r, g, b)
Creates a color matching the provided RGB values.

Values are integers and go up to 255.

Parameters:

  • r int the red value of the new color
  • g int the green value of the new color
  • b int the blue value of the new color
