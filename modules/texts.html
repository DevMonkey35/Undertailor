newText(baseParams, text)
Creates a new text object.

text objects are composed of two parts, their own parameters and their children components. The first argument for this function defines the default parameters for component text when the child parameter chooses not to set it. The only required parent parameter is the text parameter. All other parameters are optional.

Parameters:

  • baseParams string the base parameters for the text object
  • text string a string denoting all child components of the text
